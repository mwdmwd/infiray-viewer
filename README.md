# (InfiRay) Thermal image viewer

A simple utility to view and analyse InfiRay `.irg` thermal images.

## Installation

```bash
pip install .
```

## Usage

Run the viewer from the command line:
```bash
infiray-viewer [image.irg]
```

To install XDG desktop integration (menu entry and file type association):
```bash
infiray-viewer --install-desktop
```
